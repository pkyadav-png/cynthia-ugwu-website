🚀 Cynthia Ugwu Portfolio Clone

A sleek, minimalist, and highly animated portfolio website inspired by the award-winning design of Cynthia Ugwu. This project focuses on high-quality frontend interactions and smooth scrolling experiences.

✨ Key Features

. Smooth Scrolling: Integrated with Locomotive Scroll for a premium feel.

. GSAP Animations: Advanced micro-interactions and entrance animations.

. Custom Cursor: Dynamic mouse-follower effect that reacts to elements.
